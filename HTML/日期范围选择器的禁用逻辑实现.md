**方案：** 监听第一个 input 的 `change` 事件，动态设置第二个 input 的 `min` 和 `max` 属性。

```html
<div>
  <input type="date" id="startDate" />
  <span>~</span>
  <input type="date" id="endDate" />
</div>
```

```js
const startDate = document.getElementById('startDate');
const endDate = document.getElementById('endDate');

startDate.addEventListener('change', () => {
  const start = new Date(startDate.value);

  // min = 开始日期当天
  const min = startDate.value;

  // max = 开始日期 + 30天
  const maxDate = new Date(start);
  maxDate.setDate(maxDate.getDate() + 30);
  const max = maxDate.toISOString().split('T')[0];

  endDate.min = min;
  endDate.max = max;

  // 若已选的结束日期超出范围，则清空
  if (endDate.value && (endDate.value < min || endDate.value > max)) {
    endDate.value = '';
  }
});
```

**关键点：**
- `input[type=date]` 原生支持 `min` / `max` 属性，浏览器会自动禁用超出范围的日期
- `toISOString().split('T')[0]` 格式化为 `YYYY-MM-DD`，与 input value 格式一致
- 每次 start 变更时同步清空越界的 end 值，避免脏数据

```js
function formatDate(dateStr, offsetDays = 0) {
  const date = new Date(dateStr);
  date.setDate(date.getDate() + offsetDays);
  return date.toLocaleDateString('en-US', { month: 'short', day: 'numeric' })
             .replace(' ', '-'); // "Mar 5" → "Mar-5"
}

startDate.addEventListener('change', () => {
  const val = startDate.value; // "2026-03-05"

  console.log(formatDate(val, 0)); // Mar-5
  console.log(formatDate(val, 1)); // Mar-6
  console.log(formatDate(val, 2)); // Mar-7
});
```

**说明：**
- `toLocaleDateString('en-US', { month: 'short', day: 'numeric' })` 返回 `"Mar 5"` 格式
- `.replace(' ', '-')` 将空格换成 `-` 得到 `"Mar-5"`
- `offsetDays` 控制偏移天数，传 `0/1/2` 即可得到连续三天
