## 标签页管理器

一个功能强大的标签页管理工具，包含：
- 📑 查看和搜索所有标签页
- ✂️ 批量关闭标签页
- 🗂️ 创建标签页分组
- 💾 保存/恢复会话
- 🔄 自动关闭重复标签
- 📊 实时统计信息

核心技术点：
- Chrome Tabs API
- Background Script 后台任务
- 快捷键命令支持
- 通知系统集成


### 📑 标签页管理器使用指南

第一次使用

1. 打开管理器
- 点击浏览器工具栏的 📑 图标，会弹出管理面板：
```
┌────────────────────────────────┐
│  📑 智能标签页管理器          │
├────────────────────────────────┤
│  统计：总计 15  窗口 1  重复 2 │
├────────────────────────────────┤
│  [当前标签] [分组] [会话]     │
├────────────────────────────────┤
│  🔍 [搜索标签...]             │
├────────────────────────────────┤
│  标签列表...                   │
└────────────────────────────────┘
```

基础操作
- 操作 1：搜索标签
- 操作 2：激活标签
- 操作 3：关闭标签
- 操作 4：关闭重复标签
- 进阶功能：分组管理：创建、查看分组、打开分组、删除分组
- 管理会话、保存会话、
